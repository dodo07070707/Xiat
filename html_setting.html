HTML/CSS 개발환경세팅가이드 (Ver.2022.11.14)
모든 개발환경세팅가이드는 포멧된 상태를 기준으로 하고 있습니다.
Atom이 설치되어있다면 프로그램 내 세팅 칸으로 이동해주세요.

프로그램다운로드
1. atom.io에 접속 후 다운로드 (windows/mac둘다 자동으로 운영체제에 맞는 파일 다운로드 됨)
2. 다운로드 받은 파일을 압축해제
3. 압축해제 된 파일을 실행

프로그램 내 파일 세팅
4. 좌측 상단 메뉴바에서 file -> newfile을 통해 파일 생성
5. 좌측 상단 메뉴바에서 save as를 통해 파일의 이름과 확장자 설정
