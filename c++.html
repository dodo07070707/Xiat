C++
한줄소개
C++은 프로그래밍 언어의 근본이 되는 C언어를 기반으로 이루어진 언어이다.
하지만 C언어와는 달리 객체지향 언어이기 때문에 유지보수가 편리하고 생산 효율이 높다는 장점이 있다 .
모든언어들의 기본이 되는 언어로써 한번 배워두면 다른 언어들을 배울 때 많이 사용됩니다.
C++은 근본을 C언어에 두는 만큼 C언어에 대해 약간의 이해를 필요로합니다.

습득난이도
★★★☆☆

사용분야
운영체제 (윈도우,리눅스) , 웹서버, 반도체 내 CPU, 임베디드시스템 (키오스크,네비게이션,CCTV,세탁기,냉장고,전자레인지) 내의 프로그램

개발자의 한줄평
고통을 즐겨라.

여러가지 팁들
헤더파일
   #include<stdio.h>, #include<iostream>과 같은 헤더들을 귀찮게 따로따로 쓸 필요가 없다
   #include<bits/stdc++.h>를 사용하면 c++에서 사용할 수 있는 대부분의 헤더들이 포함되어 있어 보다 편리하게 코딩할 수 있다.
디버깅
   컴파일 에러가 나면 코드를 보고 고치기 전에 에러 창을 보자 . 대부분의 컴파일 에러는 에러메시지를 보면 어느 줄에서 어떤 부분이 문젠지 알려준다.
기본문법단축
   C++은 c언어와 달리 입출력을 할 때 scanf(“%d”, 변수이름) ; 형식을 사용할 필요 없이
   바로 cin >> 변수이름 ; 형식으로 간단하게 축약할 수 있고 printf(“%d”, 변수이름) ; 을 cout << 변수이름 ; 으로 축약할 수 있다 .
   using namespace std ; 헤더를 사용하면 입출력 단순화가 가능하다.
